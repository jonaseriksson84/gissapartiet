# Add a 9th "Partilös" button for politiska vildar

Politiska vildar (MPs who broke from their elected party but kept their Riksdag seat) are included in the game's pool, with the *current* affiliation per `data.riksdagen.se` as the right answer. For most vildar this is "Partilös" — which doesn't correspond to any of the eight Riksdag parties — so the UI exposes a 9th button labelled "Partilös" alongside the eight party buttons, ensuring vilde rounds are winnable.

The cost is a slightly lower random baseline (11.1% vs 12.5%) and one extra button players must skip on most rounds; both judged acceptable in exchange for honouring the principle that *current* affiliation is always the right answer.
